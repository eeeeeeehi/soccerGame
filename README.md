# soccerGame

このプロジェクトは、TypeScriptとHTML5 Canvasで作られた2Dアーケードサッカーゲームです。 現在のコードベースから読み取れるゲームの詳細は以下の通りです。

1. ゲームの概要
形式: 11人 vs 11人のフルコートサッカー。
視点: 上空からの2D視点（トップダウンビュー）。
対戦形式: プレイヤー（Team 1 / 青） vs CPU（Team 2 / 赤）。
2. 主な機能とシステム
カスタムエンジン: Phaserなどの既存のゲームエンジンを使わず、独自のゲームループ、物理演算、レンダリング処理（Canvas API）で構築されています。
選手データ: 実在の有名選手（メッシ、ロナウド、エムバペ、ハーランドなど）が登場し、それぞれの能力値（スピード、キック力、スタミナ、テクニック）やポジション適正（FW, MF, DF, GK）が設定されています。
フォーメーション: デフォルトで4-4-2などのフォーメーションがあり、各選手には「ホームポジション」が設定され、試合展開に合わせて定位置に戻る動きをします。
セットプレー:
コーナーキック: 具体的なポジショニングロジック（ニア、ファー、ショートコーナーなど）が実装されています。
スローイン: 味方がボールを受ける動きなどがプログラムされています。
ゴールキック: 実装済み。
AI（CPU）: ボールに最も近い選手がプレッシャーをかけ（チェイサー）、それ以外の選手は陣形を保つようなAIが組まれています。
3. 操作方法（コードからの推測）
移動: 矢印キー（またはWASD）で選択中の選手を操作。
選手切り替え: Space キーで守備時にボールに近い選手に操作を切り替える機能（またはスライディングなど）があります。
ボール保持: ドリブルは「吸い付き」系（Sticky Dribble）で、急ターン時などの制御ロジックが含まれています。
4. デザイン・ビジュアル
定数ファイル（Constants.ts）によると、フィールドサイズは1200x800px。
UIとしてサイドバーが表示され、スコアや各種情報が表示される設計です。
シンプルながらも、選手ごとのステータス差やセットプレーの戦術的な配置など、細かい部分まで作り込まれつつあるプロジェクトです
