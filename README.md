# ⚽ Arcade Soccer Game (HTML5 Canvas / TypeScript)

TypeScript と HTML5 Canvas を用いて開発した、  
**2Dトップダウン視点のアーケードサッカーゲーム**です。  
Phaser などの既存ゲームエンジンは使用せず、**独自のゲームループ・物理演算・AIロジック**を実装しています。

---

## 🎮 ゲーム概要

- **形式**：11人 vs 11人 フルコートサッカー  
- **視点**：トップダウン（上空視点）  
- **対戦**：プレイヤー（青チーム） vs CPU（赤チーム）  
- **ジャンル**：アーケード × サッカー（ウイイレ風）

---

## 🛠 技術スタック

- **TypeScript**
- **HTML5 Canvas**
- **Vite**（開発環境）
- 独自ゲームエンジン（Phaser / Unity 等は不使用）

---

## 🧠 システム構成・特徴

### 🎯 独自ゲームエンジン
- requestAnimationFrame ベースの **独自ゲームループ**
- Canvas API による描画
- 簡易物理演算（速度・摩擦・衝突処理）

### 👥 選手・チーム設計
- 実在選手をモデルにしたキャラクター（Messi, Ronaldo など）
- 各選手に以下の能力値を設定：
  - Speed
  - Kick Power
  - Stamina
  - Technique
- ポジション適正：
  - FW / MF / DF / GK

### 🧩 フォーメーションシステム
- 4-4-2 などの基本フォーメーションを実装
- 各選手は **Home Position（基準位置）** を保持
- ボール非関与時は陣形を維持する挙動

---

## 🤖 AI（CPU）ロジック

- **最寄り選手のみがボールへプレッシャー**
- その他の選手は：
  - フォーメーション維持
  - マーク
  - パスコースのカバー
- 全員がボールに突撃しない設計

---

## ⚽ セットプレー実装

- **コーナーキック**
  - ニア / ファー / ショートコーナーの配置ロジック
- **スローイン**
  - 味方が受けに動く簡易戦術
- **ゴールキック**
  - 再開位置・配置処理を実装

---

## 🎮 操作方法

| 操作 | キー |
|---|---|
| 移動 | 矢印キー / WASD |
| アクション（キック / タックル） | Space |
| ダッシュ | Shift |
| 選手切り替え | S |
| ヘルプ表示 | H |

- 操作中の選手は **足元に緑の円** で表示
- ドリブルは **Sticky Dribble（吸い付き）方式**

---

## 🖥 UI・ビジュアル

- フィールドサイズ：**1200 × 800 px**
- 左右にサイドバーを配置
- スコア・選手データ・マネジメント情報を表示
- シンプルながら戦術性を意識したデザ
